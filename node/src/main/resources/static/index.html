Message placeholders to replace later or in node process.

Placeholder for message parameters that are to be exchanged later.
  • {{ KEY:<key> }}
    Replace with a saved message.
  • {{ PLACEHOLDER:<placeholder> }}
    Placeholder which will be replaced later on the client side.

List pattern, handle undefined length. (Rest-Request)

To define requests for lists with an infinite amount, there are patterns that can be sent as arguments to the rest server.
  • pagination::1::5&sort::asc::name
    Will get first page with 5 elements orderd ascending by their name.
  • pagination::1::5
    Will get first page with 5 elements without order.
  • sort::desc::name
    Will order by name descending.
